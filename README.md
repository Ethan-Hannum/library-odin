# library-odin

Library Project for The Odin Project

This library will set book objects into an array and display each book on the page.
Books added to the library will not save between page refreshes.

This project helped me get a solid grasp on object constructors, how to manipulate
object data, and how objects and array can work together! Being able to set objects in array, and vice-versa will
massivley help in the future to iterate each object and to search through those object (especially with the crypto API).
